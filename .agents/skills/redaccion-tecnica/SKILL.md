# Redaccion Tecnica

## Objective

Keep technical writing specific, source-bound, and aligned with the documentary style already present in Plaza Mayor.

This skill is not a license to embellish. It exists to remove generic wording and keep scope, uncertainty, and technical grounding explicit.

## When to use

Use when:

1. Drafting or revising memory or annex prose.
2. The user says the text sounds generic or AI-written.
3. A paragraph needs tightening without changing its technical meaning.
4. Scope and pending data need to be stated clearly.

Do not use to invent missing technical content. Missing data stays missing.

## Source discipline

Before rewriting:

1. Identify the governing source for the data in the paragraph.
2. Preserve numbers, codes, and scope statements exactly unless the user asked for correction.
3. If the draft contains placeholders or pending data, keep that uncertainty visible.

## Plaza Mayor style markers

Prefer prose that:

1. States the object of the section early.
2. Delimits what is confirmed and what remains pending.
3. Uses impersonal technical Spanish instead of conversational style.
4. Avoids filler transitions and rhetorical emphasis.
5. Keeps wording compatible with project terminology such as `anejo`, `medicion`, `partida`, `importe`, `ambito`, `trazabilidad`.

## Avoid

- empty emphasis such as `cabe destacar`, `es importante senalar`, `hay que tener en cuenta`
- circular justification such as `es la solucion mas adecuada` without data
- generic admin prose that could fit any project
- long rewrites when only one sentence is bad
- changing annex numbering or section structure without request

## Rewrite method

1. Keep the technical claim.
2. Remove generic filler.
3. Make the scope boundary explicit.
4. Preserve pending items as pending.
5. Keep the paragraph proportional to the actual data available.

## Output contract

When revising existing text, report:

```text
Problema:
Motivo:
Correccion:
```

When drafting from scratch, report:

```text
Objeto:
Datos confirmados usados:
Pendientes visibles:
Texto propuesto:
```

## Stop rules

Do not finalize rewritten prose if:

1. The wording became more confident than the source allows.
2. A normative or numeric reference was paraphrased into vagueness.
3. The rewrite hides that the annex is still in `arranque documental`.
4. The paragraph now sounds polished but less traceable.
