# OUTPUTS_AI

Carpeta reservada para salidas tecnicas generadas a partir de datos del proyecto.

No es una carpeta de plantillas corporativas.
No sustituye memoria ni anejos.

Subcarpetas previstas:

- `tablas/`
- `graficos/`
- `dashboards/`
- `plan_obra/`
- `propuestas_valor/`

Reglas:

1. todo output serio debe tener manifest en `CONTROL/ai_runs/`
2. si una salida depende de hipotesis, debe indicarlo
3. si faltan datos, se deja pendiente visible
4. no guardar aqui DOCX de memoria o anejos corporativos
