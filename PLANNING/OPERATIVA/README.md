# Operativa Reubicada

Esta carpeta agrupa documentacion operativa que antes contaminaba la raiz del repo.

## Criterio

- Si el archivo explica como trabajar, pero no define el estado tecnico del expediente, no debe vivir en raiz.
- Si el archivo es una receta o una guia auxiliar, debe vivir aqui o acabar en el toolkit cuando se consolide.
