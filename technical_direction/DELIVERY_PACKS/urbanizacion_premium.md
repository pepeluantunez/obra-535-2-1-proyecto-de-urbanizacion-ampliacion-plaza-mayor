# Pack Urbanizacion Premium

## Objetivo

Definir una entrega interna premium para proyectos de urbanizacion sin tocar plantillas corporativas base.

## Obligatorio

- coherencia minima memoria-anexos-presupuesto segun fuentes maestras
- trazabilidad minima de outputs relevantes
- tabla de decisiones y pendientes
- informe de incoherencias si existen contradicciones abiertas

## Premium recomendado

- grafico de PEM por capitulos
- tabla de partidas criticas
- tabla de redes por diametro y material
- Gantt preliminar si existe base suficiente
- curva S si existe presupuesto y cronograma trazables
- dashboard resumen del proyecto
- informe de propuestas de valor anadido

## Regla

Si faltan datos para un output premium, no se elimina en silencio:

1. se deja como no generable;
2. se explica por que;
3. se registra el pendiente correspondiente en `DECISION_LOG.md` o en el manifest del output.
