# Technical Direction

Esta carpeta agrupa la capa de direccion tecnica aumentada del proyecto.

No sustituye:

- `MAPA_PROYECTO.md`
- `FUENTES_MAESTRAS.md`
- `DECISIONES_PROYECTO.md`
- `ESTADO_PROYECTO.md`

Su funcion es otra:

1. proponer mejoras premium;
2. revisar coherencia transversal;
3. registrar decisiones abiertas;
4. preparar paquetes de salida profesional sobre datos y trazabilidad.

Regla de capa:

- si una pieza resulta reutilizable entre expedientes, debe promocionarse a `urbanizacion-toolkit`;
- mientras tanto, aqui vive solo como semilla local validada.
