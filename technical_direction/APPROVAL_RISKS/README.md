# Approval Risks

Esta carpeta recoge riesgos de aprobacion o cuestionamiento del proyecto.

No son riesgos de obra.

Son riesgos de que el expediente sea objetado, devuelto, cuestionado o debilitado por:

- ayuntamiento
- companias suministradoras
- supervision tecnica
- control interno
- jefe de proyecto
- cliente

Uso:

1. registrar riesgos de aprobacion visibles;
2. proponer mitigaciones concretas;
3. alimentar la revision dura y la coherencia total.
